unflat.css

A currently experimental collection of css only widgets, use these to extend flat.css

flat.css styles HTML5 elements only, it does not provide composite components.

These are all intended to have really simple flexible HTML, HTML is more of a priority than zero javascript. For example javascript should be used to add or remove the `show` class to show or hide an element. Whilst the elements do not require javascript at the moment but may in the future. (As an example the Modal renders here fine but I can't think of a time when I've ever wanted a Modal in a non javascript enabled application)

  • More components to come

Modal

The reveal and modal classes used together can produce this component.

This modal is best when used with some javascript, more explaination required.

This example page uses reveal to show this dialog, and requires an element on the page with id="modal-closed" to close it. Instead in most applications I recommend you add some javascript to add and remove the show class. example required

Grid

A really lightweight grid, add class="row" to a class for it to be a continer. Then add class="column" to content you want to column to fill the space. Other elements will behave as columns but not be stretchy. When horizontal space is limited it will just use one or two columns.

Expand

A reveal expand will expand the content in place.

Reveal

A reveal is not intended for use on its own, this is what it does. It leaves preallocated space for the hidden content.

Menu
